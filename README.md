#PassMan

- A command line passowrd generator and manager with a lock feature that hides the password for a profile for specific period of time.
> Lock feature -> hides the password till the set time, 
 Many phone/social/internet detox apps provide pasword option for the dextox setting but we can easily disable them with password unless third party has provide the passcode this features genreates a strong password copies it to the clipboad and locks the password till set date and time, helping commit to the detox.
